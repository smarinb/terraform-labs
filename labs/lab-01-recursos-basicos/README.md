# Lab 01 - Recursos Básicos

## 🎯 Objetivo
Entender la estructura básica de Terraform: `resource`, `variable`, `output`.

## 📁 Archivos
- `main.tf` — Recurso principal (crea un archivo local)
- `variables.tf` — Variables de entrada
- `outputs.tf` — Valores de salida

## 🚀 Cómo ejecutar

```bash
terraform init
terraform plan
terraform apply
terraform destroy
```
